Mobile Credit Transfer

This project is an app that enables users to send mobile credits to others across different countries. The app simplifies the process of transferring mobile credit internationally, allowing users to support friends and family by topping up their mobile balances no matter where they are. With a user-friendly interface and secure transaction methods, the app makes it easy to send mobile credit across borders, bridging the gap between users in different locations.

Intelligent Fashion Assistant

This project is an Intelligent Fashion Assistant app that leverages machine learning to help users choose outfits tailored to their unique attributes. The app analyzes factors such as body shape, weight, skin color, gender, and other personal properties to provide personalized clothing recommendations. By using advanced algorithms, the app ensures that users can find the most flattering and suitable outfits, making fashion choices easier and more intuitive. This innovative approach to fashion combines technology with style, offering a personalized shopping experience for every user.
